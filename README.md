# 🎮 Gusick Game Studio - 묵찌빠

한국 전통 게임인 묵찌빠를 현대적인 웹 기술로 구현한 멀티플레이어 게임입니다.

## ✨ 주요 기능

- 🤖 **AI와 대전**: 똑똑한 AI와 묵찌빠 게임을 즐겨보세요
- 👥 **멀티플레이어**: 실시간으로 다른 플레이어와 대전
- 🔊 **사운드 시스템**: 게임을 더욱 재미있게 만드는 효과음과 배경음
- 📱 **반응형 디자인**: 모바일과 데스크톱 모두에서 완벽한 경험
- 🎨 **현대적인 UI**: 아름답고 직관적인 사용자 인터페이스
- 📊 **통계 시스템**: 게임 기록과 승률 추적

## 🎯 게임 규칙

### 묵찌빠란?
묵찌빠는 한국의 전통적인 손놀이 게임입니다.

1. **1단계 - 가위바위보**: 먼저 가위바위보로 공격자를 정합니다
2. **2단계 - 묵찌빠**: 공격자가 정해지면 묵찌빠를 시작합니다
   - 같은 모양을 내면 공격자가 승리
   - 다른 모양을 내면 이긴 사람이 새로운 공격자가 됩니다
3. **승리 조건**: 5라운드 중 더 많이 이긴 사람이 최종 승자

## 🚀 기술 스택

- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Animation**: Framer Motion
- **Audio**: Web Audio API
- **Storage**: Local Storage (GitHub API 연동 준비)
- **Deployment**: GitHub Pages

## 🎵 오디오 기능

- 게임 시작 음성 안내
- 버튼 클릭 효과음
- 움직임 선택 사운드
- 라운드 완료 알림음
- 승리/패배 사운드
- 음소거 및 볼륨 조절 기능

## 🎮 플레이 방법

1. **회원가입/로그인**: 게임을 시작하기 위해 계정을 만드세요
2. **게임 모드 선택**: 
   - AI와 대전: 컴퓨터와 연습하기
   - 멀티플레이어: 실시간으로 다른 플레이어와 대전
3. **게임 플레이**: 
   - 묵(✊), 찌(✌️), 빠(✋) 중 하나를 선택
   - 전략적으로 상대방의 움직임을 예측하세요
